FRONTEND TECHNOLOGIES

HTML (HyperText Markup Language)

HTML (HyperText Markup Language) is the standard language used to create and design the structure of web pages. It consists of a series of elements represented by tags, which define different parts of the content, such as headings, paragraphs, links, images, and more.

HTML was introduced by Tim Berners-Lee in 1991.

CSS(Cascading Style Sheets)

CSS is a stylesheet language that describes the presentation of HTML elements on a webpage. It separates content (HTML) from presentation, ensuring better maintainability, flexibility, and consistency across web projects.The purpose of CSS is to Styling Web Pages and Responsive Design.

BACKEND TECHOLOGIES

PYTHON

Python is a high-level, interpreted, and general-purpose programming language known for its simplicity, readability, and versatility.

Python was introduced by Guido van Rossum in 1991.

Node.js

Node.js is an open-source, cross-platform runtime environment that allows developers to run JavaScript code outside of a web browser. It is built on Chrome's V8 JavaScript engine and is primarily used for building scalable, high-performance web applications, particularly on the server-side.

Node.js was introduced by Ryan Dahl in 2009.

React.js

React.js, commonly referred to as React, is an open-source JavaScript library for building user interfaces (UIs), particularly for single-page applications (SPAs). It was developed by Facebook (now Meta) and is maintained by Facebook and a community of developers.

React.js was introduced by Jordan Walke, a software engineer at Facebook, in 2011.

DATABASE

MYSQL

MySQL is one of the most widely used relational database management systems (RDBMS).MySQL is a powerful database management system that uses SQL (Structured Query Language) for querying and managing databases.It is open-sourced.

It was originally created by MySQL AB in 1995.

Mongodb

MongoDB is a popular, open-source, NoSQL database that stores data in a flexible, document-oriented format. Unlike traditional relational databases, which store data in tables with rows and columns, MongoDB stores data in JSON-like documents using BSON (Binary JSON), making it well-suited for applications that deal with unstructured or semi-structured data.

MongoDB was introduced by 10gen, which later became MongoDB, Inc.. It was initially released in 2009 by a team led by Eliot Horowitz and Dwight Merriman.
